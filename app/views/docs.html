Docs & Documentation

Our main goal at Dride is to create a community of car & technology lovers. This is why we're created the docs to be simple and powerful.

Developing for dride is easy as developing a Node.js module, deploying is as simple as publishing to NPM. You get all the cool API's available right from the dride module.

Ready to get started?

npm install -g dride
mkdir dride-project
cd dride-project
dride start yourProjectName

To deploy your package simply use

dride publish

A list of available API's

  • Video capture
  • Voice commands & alerts
  • Microphone
  • Road vision
  • GPS
