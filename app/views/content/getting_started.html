Automated Setup

In order to set-up your raspberry Pi with your Dride, Run the script below.

You can also download a pre-installed pre-configured drideOS (Raspbian Jessie) image with all the software needed. Or use the Manual Installation Guide.



Download drideOS

drideOS

v0.1 Alpha
Learn how to install the operating system image on an SD card for Mac OS, Windows & Linux

Download & run the install script

bash -c "$(curl -s https://dride.io/code/install.sh)"

This script may take up to 3 hours so grab a cup of coffee before starting.


After the script is completed, you will have the following packages installed on your Dride:

  • OpenCV
  • pip
  • numpy
  • Python
  • omxplayer
  • hostapd
  • Cardigan (Dride-core)

Architecture
