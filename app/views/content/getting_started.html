Automated Setup

In order to set-up your raspberry Pi with your Dride, you need to download the pre-installed Raspbian Jessie image with all the software needed. You can also use the Manual Installation Guide.

Download & run the install script

wget https://dride.io/code/install.sh . install.sh

This script may take you a while so grab a cup of coffee before starting.


After the script is completed, you will have the following packages installed on your Dride:

  • OpenCV
  • pip
  • Python
  • omxplayer
  • Cardigan (Dride-core)
  • Wifi setup


Architecture

Dride is based on Node.js. It is controlled by an array of native and custom modules.

dride/ ├── package.json │ ├── node_modules/ │ ├── dride-ws/ │ ├── dride-core/ │ ├── dride-connectivity/ │ ├── dride-indicators/ │ ├── dride-cloud/ │ ├── dride-alexa/ │ └── ... ├── startup.js │ └── ...
