Automated Setup

In order to setup your raspberryPi with dride you need to download the preinstalled Raspbian Jessie image with all the software needed or use the manual installation guide:

Download & run the install script

wget https://dride.io/code/install.sh . install.sh

This script will take a while so you can grab a cup of coffee.


After the script is done we will have the following pakages insatlled on our dride:

  • OpenCV
  • pip
  • Python
  • omxplayer
  • Cardigan (Dride-core)
  • Wifi setup


Architecture

Dride is based on Node.js. It is controlled by an array of native and custom modules.

dride/ ├── package.json │ ├── node_modules/ │ ├── dride-ws/ │ ├── dride-core/ │ ├── dride-dvr/ │ ├── dride-connectivity/ │ ├── dride-indicators/ │ ├── dride-cloud/ │ ├── dride-alexa/ │ └── ... ├── startup.js │ └── ...
